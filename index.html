How to Use the Webmailto Plugin

First make sure to include the plugin inside your <head/> tag:

Then you can call webmailto on link tags in the normal jQuery way:

Hovering over any matched elements will cause an absolutely positioned list of webmail links to appear nearby. If the selector matches a link tag which is not a mailto link, or if the tag is not an <a/> tag, then it is silently ignored. In the above case, all mailto links on the page would be matched.

Here's an example of the most vanilla case, without any styling:

first@example.com second@example.com third@example.com

You'll notice in the above example that webmailto does not apply any default styling. However, you can define your own styles by implementing the webmailto class or by passing css into the plugin as optional parameters.
